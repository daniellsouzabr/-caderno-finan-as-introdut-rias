# Caderno Temático: Finanças Introdutórias

## 📌 Descrição
Este projeto consiste na criação de um caderno temático sobre **finanças pessoais introdutórias**, utilizando fontes abertas em texto e PDF.  
O objetivo é apoiar o aprendizado ativo por meio da organização de conteúdos, elaboração de perguntas estratégicas e uso de prompts de IA para revisões futuras.

## 🎯 Objetivos de Estudo
- Compreender conceitos básicos de orçamento pessoal.
- Diferenciar juros simples e compostos.
- Entender o impacto da inflação no poder de compra.
- Identificar ativos e passivos em finanças pessoais.

## ❓ Perguntas Estratégicas
- O que é orçamento pessoal e como estruturá-lo?
- Qual a diferença entre juros simples e compostos?
- Por que a inflação reduz o poder de compra?
- Como distinguir ativos de passivos?

## 🛠 Estrutura do Projeto
- `/fontes` → PDFs e textos utilizados como referência.
- `/resumos` → Resumos estruturados dos principais conceitos.
- `/glossario` → Termos e definições importantes.
- `/prompts` → Conjunto de prompts reutilizáveis para revisões.

## 📚 Resultado Esperado
- Mini guia de estudo com resumos claros.
- Glossário de conceitos financeiros básicos.
- Prompts prontos para futuras revisões e práticas.

## 🚀 Tecnologias e Ferramentas
- NotebookLM (para organização e geração de conteúdo).
- Fontes abertas (artigos, PDFs, blogs acadêmicos).
- GitHub (para versionamento e compartilhamento).
